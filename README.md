# Darsu Smart Calculator

A modern and feature-rich calculator built with HTML, CSS, JavaScript, and Tailwind CSS. This project combines a beautiful glassmorphism interface with scientific calculations, history management, memory operations, keyboard shortcuts, multiple themes, and persistent storage.

## Features

### Welcome Screen

* Dynamic greeting based on the current time
* Smooth transition into the calculator
* Interactive start button

### Modern User Interface

* Glassmorphism design
* Responsive layout for different screen sizes
* Animated floating background elements
* Smooth hover and click effects
* Ripple button animations
* Result pulse animation
* Custom scrollbar styling

### Calculator Functions

* Addition, subtraction, multiplication, and division
* Percentage calculations
* Decimal number support
* Backspace functionality
* Clear all functionality
* Real-time expression preview
* Live result preview while typing

### Scientific Calculator

* Square Root (√)
* Square (x²)
* Cube (x³)
* Power (xʸ)
* Nth Root (ʸ√x)
* Sine (sin)
* Cosine (cos)
* Tangent (tan)
* Logarithm (log)
* Natural Logarithm (ln)
* Exponential (eˣ)
* Reciprocal (1/x)
* Factorial (n!)
* Absolute Value (|x|)
* Pi (π)
* Euler's Number (e)
* Random Number Generator
* Floor Function
* Ceil Function
* Round Function

### Angle Modes

* Degree Mode (DEG)
* Radian Mode (RAD)
* One-click mode switching

### Memory System

* Memory Clear (MC)
* Memory Recall (MR)
* Memory Add (M+)
* Memory Subtract (M−)
* Persistent memory storage

### History System

* Automatically saves calculations
* Search through previous calculations
* Reuse previous results with one click
* History grouped by:

  * Today
  * Yesterday
  * This Week
  * Earlier
* Export history as a text file
* Clear history option
* Persistent history storage

### Statistics Dashboard

* Total calculations counter
* Largest calculated value tracker
* Memory value tracker
* Last usage timestamp

### Theme System

* Sky Theme
* Purple Theme
* Green Theme
* Sunset Theme
* Dark Theme
* Theme preference saved automatically

### Keyboard Shortcuts

* Number keys (0–9)
* Mathematical operators
* Enter or = for calculation
* Backspace for delete
* Escape for clear
* ? key for shortcut guide

### Utility Features

* Live digital clock
* Copy result to clipboard
* Toast notifications
* Sound effects
* Vibration feedback (supported devices)
* Keyboard key highlighting
* Error handling for invalid expressions
* Division by zero protection
* Negative square root validation

### Data Persistence

* Saves calculator state automatically
* Stores:

  * History
  * Themes
  * Memory values
  * Statistics
  * Angle mode
* Restores data when the application is reopened

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Tailwind CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/darsu-smart-calculator.git
```

Navigate to the project folder:

```bash
cd darsu-smart-calculator
```

Open `index.html` in your browser.
